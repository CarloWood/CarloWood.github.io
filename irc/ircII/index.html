Undernet IRC II scripts

Download all of the following scripts into a directory called `unet' in your default ircII scripts directory.
Or extract this tar file in your ircII scripts directory.

Then add to your .ircrc file the line: 

SET NOVICE OFF
SET LOAD_PATH <the directory where unet/ resides>
LOAD unet/extensions

  • unet/extensions
    		•  This script loads all of the other scripts that are needed.
  • unet/commands
    		•  This script contains the ALIASes for all new or extented commands.
  • unet/opercommands
    		•  This script contains the ALIASes for all new or extented commands for IRC Operators.
  • unet/numerics
    		•  This scripts handles the formatting of new or extended numeric server replies.
  • unet/layout
    		•  This script handles the formatting of `old' numerics.
  • unet/stripu
    		•  This script contains a utility which abbreviates the rather long server names by replacing redundant parts with a star.
  • unet/strlen
    		•  This script contains a utility that calculates the length of a given string.
  • unet/getname
    		•  This script contains a utility that extracts the name of a client name.
