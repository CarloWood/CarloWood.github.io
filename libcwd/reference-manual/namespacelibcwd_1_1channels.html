 dc
 This namespace contains the standard debug channels of libcwd.
 

Detailed Description

The default DEBUGCHANNELS namespace.

This macro should contain the namespace where your application defines the dc namespace; If you want to define additional debug channels, next to the ones already provided by libcwd, then the macro DEBUGCHANNELS must be defined in the custom debug.h header file of your application (or library) prior to including <libcwd/debug.h>.
