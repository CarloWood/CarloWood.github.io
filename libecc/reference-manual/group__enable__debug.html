–enable-debug (ECC_DEBUG)
Configuration, Installation And Getting Started » Configuration Options And Macros
Collaboration diagram for –enable-debug (ECC_DEBUG):
Description

Turns on internal consistency checks of libecc.  The use of these checks can slow down libecc and are mainly intended for the maintainer of libecc.

When to use it

When you think you found a bug in libecc.

Configure option

This is disabled by default.  Use \n\n configure –enable-debug \n\n to enable it.
