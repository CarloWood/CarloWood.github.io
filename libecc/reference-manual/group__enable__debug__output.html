–enable-debug-output (ECC_DEBUGOUTPUT)
Configuration, Installation And Getting Started » Configuration Options And Macros
Collaboration diagram for –enable-debug-output (ECC_DEBUGOUTPUT):
Description

When –enable-debug (ECC_DEBUG) is enabled, this turns on a lot of extra debugging output concerning the internal state of libecc. 

When to use it

Never, unless you are a developer of libecc.

Configure option

This is disabled by default.  Use \n\n configure –enable-debug-output \n\n to enable it.
