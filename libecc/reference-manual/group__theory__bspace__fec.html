Frobenius equivalence classes visualized
This page contains the output of testsuite/point/bspacetables2.cc. Each left-most column contains all elements of the field F2m that is not part of a proper subfield (that is, the smallest positive k such that x2k = x is k = m), also known as Frobenius equivalence classes of size m. Note that the phrase "Frobenius equivalence class" is an invention of myself— you might not find it in literature [ It turns out that in existing literature one speaks preferably about Lyndon words if length m. ]Elements in the same Frobenius equivalence class have the same color. Each element appears twice: in the left-most column and in the column containing the inverses. Also this column is colored, using the same color for the same element.A few tables with m a multiple of 4, only showing the entries with m/d = 4.
