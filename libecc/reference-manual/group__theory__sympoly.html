Newton and Elementary Symmetrical Polynomials

For a fixed n, the set { S1, S2, ..., Sn } is linearly independend; they cannot be expressed into eachother. Since there are n indeterminates and n elementary symmetric polynomials, they form a basis for the set of symmetric polynomials of n indeterminates. The same holds for the set { N1, N2, ..., Nn }. Every symmetric polynomial can be expressed as a polynomial of the ESP's or NSP's, in particular they can be expressed into eachother.A recursive formula that expresses one into the other still looks reasonably simple.
