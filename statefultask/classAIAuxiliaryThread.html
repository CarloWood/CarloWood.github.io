AIAuxiliaryThread Class Reference

Detailed Description

A singleton interface to start and stop gAuxiliaryThreadEngine.

The gAuxiliaryThreadEngine must be started early, and stopped before cleanup destruction, by calling the static start and stop member functions of this class.

The real AIEngine (gAuxiliaryThreadEngine) is hidden from the user: this object is not an AIEngine (most notably, there is no need to call gAuxiliaryThreadEngine.mainloop() in a loop somewhere).

Member Function Documentation

void AIAuxiliaryThread::start ( )
static

Start the auxiliary thread.

Call this from main() before any task is run.

void AIAuxiliaryThread::stop ( )
static

Stop the auxiliary thread.

Call this before destructing objects that might be used by (call backs of) still running tasks for a clean destruction (avoiding running tasks doing call backs of destructed objects).

After this no new task may be (re)started anymore.
