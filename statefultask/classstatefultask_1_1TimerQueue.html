statefultask::TimerQueue Class Reference

Detailed Description

A queue of running (possibly cancelled) timers, all of the same interval.

This queue stores Timer*'s. Each Timer will have the same interval (which interval that is depends on the context in which the TimerQueue was found). If a pointer is nullptr then it represents a cancelled timer; such timers are not removed from the queue because that would cost too much CPU.

In the description of the member functions, 'current' means the next timer that will be returned by pop(), also if that timer was already cancelled!
